# Iris-Flower-Classification-ML
Machine Learning project using Decision Tree and Logistic Regression for Iris flower classification.

## Overview

This project classifies Iris flower species using Machine Learning algorithms.

The dataset contains measurements of:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

The target variable is the flower species:

* Iris-setosa
* Iris-versicolor
* Iris-virginica

## Algorithms Used

1. Decision Tree Classifier
2. Logistic Regression

## Features

* Data Loading and Preprocessing
* Exploratory Data Analysis (EDA)
* Pair Plot Visualization
* Correlation Heatmap
* Model Training
* Accuracy Comparison
* Confusion Matrix Analysis
* Species Prediction using User Input

## Technologies Used

* Python
* Pandas

* Matplotlib
* Seaborn

## Results

The models were evaluated using:

* Accuracy Score

* Classification Report
